# Egg Game (JavaScript)

A browser-based mini-game involving falling eggs.

## 🧰 Tech

Open `index.html` in your browser and catch the eggs using the basket.

- Falling egg animation
- Score tracking (optional)
- Fun for a quick break

Performance may be choppy; this is a prototype.
